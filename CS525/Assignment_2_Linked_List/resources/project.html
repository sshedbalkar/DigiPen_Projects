Programming assignment - linked lists

FAQ:

Most of you already have an implementation of linked list - you may reuse it for this assignment. Suggested modifications:
  • check the pointer arithmetic - I'll use CodeGuard, which is very strict. Here is an example, CodeGuard doesn't like the following code 
    int main() {
  int array[10];
  for (int *p=array;p<array;i+=6) {
    *p=0;
  }
}
    the reason is that during the second iteration p=array+12, so p points to outside of the array (remember the rule if you start with a pointer to some element in an array, the result of pointer arithmetic should be a pointer to an element of the SAME array, OR a pointer to a ONE PAST LAST). What we got in the previous example was 3 past last. The next example is OK: 
    int main() {
  int array[10];
  for (int *p=array;p<array;i+=1) {
    *p=0;
  }
}
    The best way to make sure that your pointer arithmetic is legal is to use only != when testing for loop-termination (compare to STL!): 
    int main() {
  int array[10];
  for (int *p=array;p != array;i+=5) {
    *p=0;
  }
}
  • implement one of insert technics discussed in "Linked Lists" slides, for example with a dummy node - it should simplify your code a lot.
